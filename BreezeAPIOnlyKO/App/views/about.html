About this sample

This sample was created to show off Breeze's ability to work with Knockout in a database free application.

Please feel free to use this sample to understand how to create your own application consuming any API you can find!

What is the purpose of this app? -

This sample is based on Breeze.js and Knockout.js and showcasing how quickly you can consume any API.

The main purpose of this sample is to retrieve, cache, and relate the data using Breeze.js without a database.

Why the additional dependencies? -

This sample also uses Durandal.js, Require.js, Toastr.js and ASP.NET MVC to showcase how quickly this can be done.

This sample consumes the ESPN Major League Baseball API that is publically available from their Developer's website. This sample is an open source project maintained independent of ESPN.
